# Number Guessing Game

A simple number guessing game built using HTML and JavaScript.  
The player tries to guess a randomly generated number within a fixed range.

## Features
- Random number generation between a given range
- User input using prompt
- Input validation for invalid or out-of-range values
- Tracks number of attempts
- Displays feedback for each guess (too high / too low)

## Technologies Used
- HTML
- JavaScript

## How It Works
1. The program generates a random number between 50 and 100
2. The user is prompted to guess the number
3. The program provides feedback until the correct number is guessed
4. The total number of attempts is displayed at the end
